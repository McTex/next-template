## Usage

### Install dependencies

```bash
npm run install
# or
yarn install
```

### ローカルサーバー立ち上げ

```bash
npm run dev
# or
yarn dev
```

### 静的データ書き出し

```bash
npm run generate
# or
yarn generate
```
